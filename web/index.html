free apigen your sources!

apigenerator.org creates your api documentation, using apigen just in time and push it back into your repository as github page!

It is free, try it out!

Use

Use apigenerator.org is really easy and done in three simple steps!

Step 1 create an apigen.yml in you project.

~

Yes, if you want to use default settings, just add ~ to your apigen.yml

Have a look into defaults.yml.dist for all possible options.

Our defaults are:

allowed-html: b,i,a,ul,ol,li,p,br,var,samp,kbd,tt,table,thead,tbody,tr,td,pre,code
autocomplete: classes,constants,functions,methods,properties,classconstants
access-levels: public,protected,private
internal: yes
php: false
deprecated: true
todo: true

Step 2 Allow user apigenerator to push to your repository.

for personal accounts go to project Settings → Collaborators and add apigenerator.

for organizations without apigenerator team go to organizations team page, create a new team called apigenerator, grant Push & Pull access, add apigenerator as a member and the repositories you want to generate docs for.

for organizations with apigenerator team go to project Settings → Teams and add the team apigenerator.

Security hint: We promise that we will use the access only to push the generated docs. You can check our code on github. If this is not enough for you, use the code and install your own apigenerator.org instance.

Step 3 Register a service hook.

Go to project Settings → Service Hooks → WebHook URLs and add http://apigenerator.org/hook.php

We will provide a GitHub Service Hook shortly!
